## üéØ Challenge Objectives

### Core Mission
Build applications showcasing FIBO's unique capabilities:
- **JSON-native generation** instead of text prompt engineering
- **Deterministic control** over camera angles, FOV, lighting, color palettes, composition
- **Structured parameters** that work consistently
- **Production workflows** for professional creative teams

### What Makes FIBO Different
- Trained on 1+ billion fully licensed images
- Full commercial indemnity
- JSON-structured parameter control
- Professional-grade visual parameters
- Deterministic and reproducible outputs

## üìã Project Requirements

### Technical Requirements
1. **FIBO Integration**: Must use Bria FIBO model/API
2. **Functionality**: Project must install and run consistently
3. **Platform Compatibility**: Must work on specified platforms
4. **Innovation**: Either new project or significant updates during submission period
5. **Third-party Compliance**: Authorized use of any external SDKs/APIs
6. **Ethics**: Compliance with AI ethics and safety guidelines

### Submission Requirements
1. **Public Repository**: Complete source code with setup instructions
2. **FIBO Usage**: Clear indication of model use (README, examples, API integration)
3. **Open Source License**: Apache 2.0 or MIT encouraged (not required)
4. **Project Description**: Comprehensive feature and functionality explanation
5. **Demo Video**: Under 3 minutes, publicly viewable (YouTube/Vimeo/etc.)
6. **Category Selection**: Choose best-fit category with explanation
7. **Optional**: Deployed demo or interactive experience links

### Video Requirements
- **Duration**: Maximum 3 minutes
- **Platform**: YouTube, Vimeo, Facebook Video, or Youku
- **Content**: Show project functioning on target device
- **Rights**: No copyrighted material without permission
- **Language**: English or with English translation

## ‚öñÔ∏è Judging Process

### Stage 1: Viability Check
Pass/fail determination based on:
- Project reasonably fits theme
- Reasonable application of required APIs/SDKs

### Stage 2: Scoring (Equal Weight)
1. **Usage of Bria FIBO**
   - Effective leverage of FIBO's unique capabilities
   - Showcase of JSON-native generation, professional parameters, controllability
   - Technical implementation quality

2. **Potential Impact**
   - Value for professional creative workflows
   - Real production problem solutions
   - Enterprise scalability potential
   - Impact on professional content creation

3. **Innovation & Creativity**
   - Novelty of application/approach
   - Unexpected feature combinations
   - New possibilities for structured generation
   - Significant improvement over existing solutions

### Tie Breaking
1. Highest score on first criterion (Usage of Bria FIBO)
2. If still tied, next criterion (Potential Impact)
3. If still tied, judges vote
